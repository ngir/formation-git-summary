La commande git status permet de visualiser les différences entre la HEAD et l'index.
Les fichiers et répertoires ajoutés/modifiés/supprilés sont listés.
La première partie de la liste contient les éléments qui seront committés si la commande "commit" est effectuée, la deuxième partie contient les 
éléments qui doivent d'abord être ajoutés via la commande "add".
